Help

Do you need to make SimpleStats work even better for you? SimpleStats has several options that can be customized to make it as usable and accessible as possible. Feel free to change any of the settings below. An explanation of what all of the settings mean is at the bottom of the page.

Settings

What do these setting mean?

  • Dark Mode

    Toggling the dark mode option will change the color scheme of the website so it uses darker colors. This can be easier on the eyes, especially at nighttime. By default, if your operating system's default app mode is set to dark, then so will the website.

  • Font

    The font determines how all of the text on the website appears. If you do not like the default font, feel free to change it to any of the others that look better!

  • Font Size

    The font size determines how large all of the text on the website appears. If you feel like the font size is too small or too large, feel free to change it so it looks better to you!
